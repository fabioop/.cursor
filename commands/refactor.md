# Refactor Code

Improve the selected code without changing its behavior.

> **Related Rule**: `code-quality/RULE.md`

## Rules

1. Make small, incremental changes
2. Preserve existing behavior exactly
3. Ensure tests still pass
4. Explain each refactoring step
5. Do NOT add new features

## Output Format

For each change:

### Before

```javascript
// Original code
```

### After

```javascript
// Refactored code
```

### Why

Explanation of why this improves the code.
